## Non-negotiable product boundaries
- Binance Spot only
- No futures
- No margin
- No leverage
- No withdrawal functionality
- Live trading must be disabled by default
- Paper trading must be the default runtime mode
- Every live trade must pass through the risk engine
- Every live trade must include stop-loss protection
- A kill switch must exist and be enforceable
- Unsafe behavior must be impossible by default
